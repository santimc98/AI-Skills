# Excel Analysis Skill

## Cuándo usar esta skill

Usar esta skill cuando la tarea implique analizar, limpiar, transformar, comparar o explicar datos en Excel, CSV o tablas similares.

## Objetivo

Convertir datos tabulares en conclusiones claras, métricas verificables y entregables listos para usar en reporting, Confluence, SharePoint o presentaciones.

## Entradas necesarias

- Archivo Excel, CSV o tabla.
- Objetivo del análisis.
- Hoja o rango relevante, si aplica.
- Métricas o KPIs esperados, si existen.
- Audiencia del resultado: técnica, negocio, dirección o equipo operativo.

## Procedimiento paso a paso

1. Identificar hojas, tablas, columnas y tipos de datos.
2. Entender el objetivo del usuario antes de calcular.
3. Detectar columnas clave, campos de fecha, identificadores, importes, categorías y métricas.
4. Revisar problemas básicos: nulos, duplicados, formatos inconsistentes, valores extremos y fórmulas sospechosas.
5. Calcular métricas de forma trazable.
6. Separar resultados observados de interpretaciones.
7. Preparar tablas resumen o explicación ejecutiva según el destino.
8. Si el resultado se usará en Confluence o SharePoint, entregar texto estructurado y fácil de copiar.

## Criterios de calidad

- Los cálculos deben ser reproducibles.
- No se deben mezclar supuestos con datos confirmados.
- Las conclusiones deben estar conectadas con los datos.
- Los nombres de columnas deben respetarse exactamente.
- Si hay errores o limitaciones en el archivo, deben indicarse.

## Formato de salida esperado

Según la tarea, entregar uno o varios bloques:

- Resumen ejecutivo.
- Diagnóstico de datos.
- Tabla de métricas.
- Explicación paso a paso.
- Recomendaciones.
- Fórmulas o pasos para replicarlo en Excel.

## Errores comunes a evitar

- Inventar columnas que no existen.
- Calcular medias o totales sin explicar qué datos se han excluido.
- Ignorar filtros aplicados.
- No distinguir entre hoja, tabla y rango.
- Dar una conclusión ejecutiva sin revisar la calidad del dato.
