# Assets para Educa Edtech PPTX

Esta carpeta está reservada para la plantilla corporativa `template.pptx`.

La skill original de Claude incluía un archivo binario `assets/template.pptx`. En este repositorio se ha creado la estructura y los scripts, pero la plantilla binaria debe subirse manualmente si el conector no permite subir binarios directamente.

## Archivo esperado

```text
skills/educa-edtech-pptx/assets/template.pptx
```

## Uso

La skill `skills/educa-edtech-pptx/SKILL.md` indica que las presentaciones corporativas de Educa Edtech deben partir siempre de esta plantilla, no de una presentación en blanco.
