# Examples

Carpeta para guardar ejemplos de uso de las skills.

## Qué incluir aquí

- Ejemplos de buenas páginas de Confluence.
- Ejemplos de issues de Jira bien redactados.
- Ejemplos de análisis Excel.
- Ejemplos de informes ejecutivos.
- Ejemplos de prompts o instrucciones de agentes.

## Recomendación

Evitar subir información sensible, credenciales, datos personales o documentos internos confidenciales si el repositorio es público.
