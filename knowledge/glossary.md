# Glossary

Glosario vivo de términos usados en las skills y documentación.

## IA

- **Agente:** sistema de IA diseñado para ejecutar tareas siguiendo instrucciones y usando herramientas.
- **MCP:** Model Context Protocol, protocolo para conectar modelos o agentes con herramientas y fuentes de datos externas.
- **Skill:** conjunto reutilizable de instrucciones, procedimientos, plantillas y conocimiento para realizar una tarea.
- **Prompt:** instrucción dada a un modelo de IA para guiar su comportamiento.

## Gestión de trabajo

- **Épica:** agrupación amplia de trabajo relacionado.
- **Historia de usuario:** descripción de una necesidad desde la perspectiva del usuario o negocio.
- **Criterio de aceptación:** condición verificable que debe cumplirse para considerar completado un trabajo.
- **Backlog:** lista priorizada de trabajos pendientes.

## Datos y reporting

- **KPI:** indicador clave de rendimiento.
- **Dashboard:** vista visual de métricas relevantes.
- **Dato confirmado:** información procedente de una fuente proporcionada o verificable.
- **Inferencia:** conclusión razonada a partir de datos disponibles.
